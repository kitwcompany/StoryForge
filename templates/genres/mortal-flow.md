# 凡人流 (Mortal Flow)

## 核心基调
主角是毫无背景的普通人，没有逆天灵根、没有显赫家世，仅凭谨慎、毅力和一点点机缘在修仙界步步为营。强调"苟"与"稳"，战斗靠准备而非开挂。

## 节奏策略
- **开篇**：主角资质平庸甚至低下，资源极度匮乏
- **升级节奏**：极慢，每个小境界都需大量积累，同阶无敌但越阶极难
- **爽点分布**：以弱胜强（靠准备和算计）、捡漏（别人看不上的机缘）、长寿
- **转折节奏**：被迫暴露实力、寿元危机、大势力碾压

## 反模式 (Anti-AI Patterns)
- [ ] 挂羊头卖狗肉（名义凡人流，实际还是开挂）
- [ ] 运气逆天（走哪都能捡到宝）
- [ ] 升级加速（中期开始升级速度赶上天才流）
- [ ] 智商碾压（所有对手都是傻子）
- [ ] 毫无温情（为了苟道六亲不认）

## 典型结构
1. **底层挣扎**（灵根低下，入门都困难）
2. **另辟蹊径**（发现适合凡人的修炼方法）
3. **低调发育**（默默积累，不显山露水）
4. **被迫出手**（隐忍到极限后的爆发）
5. **大势裹挟**（无法避免的宗门/时代大劫）
6. **以量取胜**（用时间磨平天赋差距）
