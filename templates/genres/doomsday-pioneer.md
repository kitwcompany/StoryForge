# 末日拓荒 (Doomsday Pioneer)

## 核心基调
末日后的世界不仅仅是生存，更是向未知荒野进发的开拓史。主角带领幸存者离开残破的城市废墟，向未被探索的荒原、辐射区、变异地带进军，建立前哨站、开辟商路、绘制新地图。强调"探索未知"的冒险感与"筚路蓝缕"的建设感并重，在废土之上重建文明的火种。

## 节奏策略
- **开篇**：主角在末日废墟中觉醒拓荒本能，意识到拥挤破败的城市遗迹已无未来，决定向荒野进发
- **升级节奏**：以探索区域扩大/前哨站建立/拓荒队伍壮大为成长线，每次发现新区域都是一次小高潮
- **爽点分布**：发现宜居谷地、打通危险商路、收服变异生物作为劳力、绘制完整区域地图、建立跨区贸易网络
- **转折节奏**：未知区域的恐怖生态、旧时代遗留的致命陷阱、拓荒者与保守派的冲突、其他拓荒队的恶性竞争

## 反模式 (Anti-Patterns)
- [ ] 探索无代价（随意进入辐射区/变异地带毫无防护与后果）
- [ ] 地图空洞（新区域只有名字没有独特生态与地貌描写）
- [ ] 建设速成（一天建立坚固前哨站，无视材料与工时）
- [ ] 路线随意（毫无规划地乱走都能发现完美定居点）
- [ ] 土著友好（所有本地势力见到拓荒者就主动合作，无利益博弈）
- [ ] 城市依赖（名为拓荒，实则始终围绕旧城市废墟打转）

## 典型结构
1. **废墟觉醒**（在城市废墟中意识到内卷无出路，必须向外开拓）
2. **踏上征途**（组建拓荒小队，筹备物资，离开安全区向荒野进发）
3. **首个前哨**（在荒野中克服环境困难，建立第一个生存据点）
4. **区域探索**（向周边未知区域派遣侦察队，绘制地图，标记资源点）
5. **商路开辟**（打通与其他幸存者聚落的贸易路线，建立物资流通）
6. **新家园**（建立稳定的拓荒领地，形成末日中的文明灯塔，吸引移民）
