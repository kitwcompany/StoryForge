# 洪荒 (Primordial/Honghuang)

## 核心基调
以中国古代神话中的洪荒时代为背景，融合《封神演义》《山海经》元素。圣人博弈、量劫轮回、天道算计为核心矛盾。

## 节奏策略
- **开篇**：以先天神灵/大能转世/人族先民身份切入，快速卷入大势
- **升级节奏**：以修行境界（地仙→天仙→金仙→大罗→准圣→圣人）为线
- **爽点分布**：获得先天灵宝、拜入圣人门下、参与封神量劫、逆天改命
- **转折节奏**：量劫降临、圣人博弈、天道算计、因果纠缠

## 反模式 (Anti-Patterns)
- [ ] 圣人降智（至高无上的圣人像村夫吵架）
- [ ] 因果无视（随意改变历史，毫无业力）
- [ ] 法宝泛滥（先天灵宝随手可得）
- [ ] 设定混乱（佛道神话胡乱混搭）
- [ ] 主角无敌（刚出场就吊打准圣）

## 典型结构
1. **洪荒初生**（在洪荒世界获得身份）
2. **拜师入门**（拜入大能门下，获得传承）
3. **初涉量劫**（参与封神/巫妖大战）
4. **积累功德**（在量劫中保全自身，积累气运）
5. **准圣之路**（斩尸证道，成为顶尖大能）
6. **圣人博弈**（与圣人掰手腕，改变天道走向）
