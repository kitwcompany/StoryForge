# 赛博朋克 (Cyberpunk)

## 核心基调
高科技与低生活的强烈反差，巨型企业掌控一切，主角在霓虹与阴影交织的都市中挣扎求生。强调义体改造、黑客技术、底层反抗。

## 节奏策略
- **开篇**：展现赛博朋克世界观，主角处于社会底层但有过人之处
- **升级节奏**：义体升级/黑客技术提升/社会地位攀升三线并进
- **爽点分布**：黑客入侵、义体战斗、企业阴谋揭露、底层逆袭
- **转折节奏**：义体排斥反应、被企业盯上、盟友是内鬼

## 反模式 (Anti-Patterns)
- [ ] 科技万能（黑客无所不能，无视系统安全）
- [ ] 底层失真（主角虽称底层但花钱如流水）
- [ ] 企业弱智（巨头公司被一人玩弄）
- [ ] 反乌托邦儿戏（推翻企业过于轻松）
- [ ] 义体无代价（随意改造，毫无副作用）

## 典型结构
1. **底层牛马**（主角是赛博世界的底层打工仔/佣兵）
2. **意外卷入**（接了一个改变命运的委托）
3. **技术升级**（获得高级义体/黑客工具）
4. **对抗企业**（与巨型企业势力交锋）
5. **阴谋核心**（发现影响世界的巨大阴谋）
6. **燎原之火**（点燃反抗之火，改变时代）
