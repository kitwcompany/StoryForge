# 诡异流 (Weird/Uncanny)

## 核心基调
以"不可名状的恐怖"为核心，强调规则类怪谈、认知污染、模因危害。与传统恐怖不同，诡异流的恐怖来自"无法理解"而非"血腥暴力"。

## 节奏策略
- **开篇**：第一条诡异规则出现，主角必须在遵守规则的前提下求生
- **升级节奏**：以"诡异事件"为单元，每个事件揭示更多世界真相
- **爽点分布**：破解规则漏洞、在绝境中找到生路、诡异能力的反向利用
- **转折节奏**：规则矛盾、自身逐渐被污染、发现诡异源头

## 反模式 (Anti-Patterns)
- [ ] 规则随意（规则可以任意打破，无严肃性）
- [ ] 解释过度（把不可名状的东西解释得一清二楚）
- [ ] 恐怖不足（诡异事件写成游戏关卡）
- [ ] 主角免疫（主角完全不受诡异影响）
- [ ] 重复套路（每个诡异都是换皮规则）

## 典型结构
1. **诡异降临**（世界规则突变，诡异出现）
2. **规则学习**（摸索诡异规律，勉强求生）
3. **能力觉醒**（获得对抗/利用诡异的能力）
4. **事件深入**（参与更危险的诡异事件）
5. **污染加剧**（自身逐渐被诡异同化）
6. **真相与抉择**（面对诡异的源头，做出选择）
