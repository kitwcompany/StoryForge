# 修仙 (Cultivation)

## 核心基调
以"长生大道"为终极目标，强调"法财侣地"四要素。与仙侠相比更偏重于修炼体系本身，境界划分更细，丹药、法宝、功法系统更复杂。

## 节奏策略
- **开篇**：确立修炼资质（灵根/体质/悟性），快速进入第一个境界
- **升级节奏**：炼气→筑基→金丹→元婴→化神→炼虚→合体→大乘→渡劫，每个大境界分前中后圆满
- **爽点分布**：突破瓶颈、炼制高阶丹药、收服灵兽、秘境夺宝
- **转折节奏**：天劫、心魔劫、寿元危机、功法缺陷

## 反模式 (Anti-Patterns)
- [ ] 境界通胀（后期境界数量失控，战力崩坏）
- [ ] 丹药万能（无视瓶颈，嗑药就能升级）
- [ ] 法宝泛滥（神器随手可得，毫无稀有感）
- [ ] 灵石贬值（前期珍贵后期废纸）
- [ ] 天劫儿戏（本该九死一生的天劫轻松度过）

## 典型结构
1. **测灵根**（确立资质，决定起点）
2. **入门修炼**（基础功法，第一次突破）
3. **外出历练**（生死搏杀，获得机缘）
4. **宗门大比**（展现实力，获得资源）
5. **结丹/元婴**（质变节点，成为强者）
6. **飞升仙界**（进入更高位面）
