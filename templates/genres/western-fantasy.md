# 西幻 (Western Fantasy)

## 核心基调
以D&D/托尔金式奇幻为基础，剑与魔法、龙与地下城、精灵与矮人。强调冒险小队、史诗 quest、善恶对决。

## 节奏策略
- **开篇**：快速建立奇幻种族和职业体系，主角获得冒险契机
- **升级节奏**：以职业等级/魔法掌握/装备获取为成长线
- **爽点分布**：新魔法习得、击败巨龙、获得传奇装备、拯救王国
- **转折节奏**：队友背叛、魔王真实身份、古老预言的反噬

## 反模式 (Anti-Patterns)
- [ ] 种族刻板（精灵全美、矮人全暴躁）
- [ ] 魔法无限制（魔力无限，随意施法）
- [ ] 冒险流水账（迷宫一层接一层，无变化）
- [ ] 魔王儿戏（最终BOSS毫无压迫感）
- [ ] 设定堆砌（大量原创名词无解释）

## 典型结构
1. **平凡出身**（村庄/孤儿/没落贵族）
2. **冒险启程**（离开故乡，加入冒险者公会）
3. **小队集结**（结识不同种族/职业的伙伴）
4. **成长历练**（完成委托，提升实力）
5. **黑暗降临**（魔王势力崛起，世界危机）
6. **最终决战**（击败魔王，成为传奇）
