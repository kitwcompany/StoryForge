# 规则怪谈 (Rules Creepypasta)

## 核心基调
以"规则文本"为核心恐怖机制的独立流派。主角必须在充满矛盾与陷阱的规则约束下求生，恐怖不来自血腥暴力，而来自"必须遵守的规则本身"。强调逻辑推理、细节观察和在规则夹缝中寻找生路。

## 节奏策略
- **开篇**：第一条规则降临，主角必须立即在遵守规则的前提下行动
- **升级节奏**：以"规则场景"为单元，每个场景的规则数量和复杂度递增
- **爽点分布**：发现规则之间的矛盾/漏洞、推理出隐藏规则、在绝境中找到合法生路
- **转折节奏**：规则被篡改、自身逐渐被规则污染、发现规则的制定者

## 反模式 (Anti-Patterns)
- [ ] 规则随意修改（规则可以任意打破或事后更改，失去严肃性）
- [ ] 恐怖廉价化（靠jump scare和血腥画面，而非规则本身的压迫感）
- [ ] 主角免疫（主角完全不受规则影响，或违反规则无后果）
- [ ] 推理跳步（逻辑链条断裂，强行发现规则漏洞）
- [ ] 规则无逻辑（规则设计杂乱无章，纯为刁难而刁难）
- [ ] 解释过度（把规则背后的恐怖存在解释得一清二楚，失去未知感）

## 典型结构
1. **规则降临**（主角进入规则领域，收到第一份规则文本）
2. **试错求生**（在遵守规则的前提下探索环境，遭遇第一次危机）
3. **规则破解**（发现规则之间的矛盾或隐藏条款，找到生路）
4. **污染加深**（主角逐渐发现规则对自己的影响，身体/精神开始异变）
5. **真相揭露**（接近规则制定者，发现规则怪谈背后的终极真相）
