# 模拟器流 (Simulator)

## 核心基调
系统流的独立分支。主角获得"人生模拟器"，可以在虚拟空间中推演不同选择的人生轨迹，获取未来信息、技能经验或特殊奖励带回现实。核心爽点是"试错权"——用无数次虚拟死亡换取现实中的一次正确选择。

## 节奏策略
- **开篇**：模拟器到账，第一次模拟揭示未来的残酷/机遇，建立信息差优势
- **升级节奏**：以模拟次数/模拟深度/可带回奖励为成长线
- **爽点分布**：模拟中发现隐藏剧情、选择最优路线、模拟中获得的技能/记忆/道具带回现实
- **转折节奏**：模拟中的死亡越来越真实（精神创伤）、模拟选项被操纵、发现模拟器真相

## 反模式 (Anti-Patterns)
- [ ] 模拟无代价（无限次模拟，毫无精神和时间成本）
- [ ] 选项无意义（看似有分支，实则只有一条正确路线）
- [ ] 蝴蝶效应缺失（模拟中改变选择，但其他事件完全不变）
- [ ] 奖励通胀（前期带回神级技能，后期无物可给）
- [ ] 现实与模拟割裂（模拟中的经验回到现实毫无违和感地完美应用）
- [ ] 模拟万能（任何问题靠模拟就能解决，无需现实努力）

## 参考数据表
| 元素 | 建议比例 | 说明 |
|------|----------|------|
| 模拟推演 | 30% | 选择分支、观察后果、总结经验 |
| 现实应用 | 25% | 将模拟获得的信息/技能用于现实 |
| 剧情发现 | 20% | 模拟中触发的隐藏事件和秘密 |
| 精神代价 | 15% | 模拟死亡的创伤、人格分裂风险 |
| 系统真相 | 10% | 模拟器的来源、更高维度的观察者 |

## 典型结构
1. **模拟器觉醒**（主角获得模拟器，首次模拟揭示未来危机）
2. **首次推演**（在模拟中尝试不同选择，积累信息差）
3. **蝴蝶效应**（现实因模拟信息而改变，但产生新的意外变量）
4. **能力带回**（模拟中获得的技能/记忆开始在现实中发挥作用）
5. **真相揭露**（发现模拟器并非简单的系统，背后有更深的存在或阴谋）
